The OpenTraffic is a free, global traffic speed data set linked to OpenStreetMap built using open-source software.

OpenTraffic operates nonprofit organization coordinating open source software development and data collection activities by individuals and organizations across the transport sector.

We are building a consortium that links connected vehicles and location-based mobile services with software and data storage platforms to annonymize and share this data as traffic statics. We also support the development of analysis and routing tools that enable cities, businesses, and individuals to make use of this data.

How it works

Our data pool connects anyone with real-time or archived location data to processing technology, data storage, and routing and analysis applications.

Conscious of location data privacy, we allow contributors to share anonymized traffic speed statistcs from derived from GPS data without disclosing location information. In return, data contributors help build a global traffic speed data set that can be used in routing and analysis applications.

The OpenTraffic data pool is comprised of several components to make the process seamless and efficient:

GPS Probes

Any real-time or historical stream of GPS data can be converted to annonymized traffic statistics
