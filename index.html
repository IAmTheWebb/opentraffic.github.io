OpenTraffic is a nonprofit organization developing real-time traffic data on a global open-source platform. We are utilizing the latest advances in open transportation technology to provide access to traffic data for cities, businesses, and individuals.

Today, anyone who needs traffic data is limited to buying a commercial set â€“ the costs of which can be a huge barrier. OpenTraffic is changing that by building a non-commercial, global traffic speed data set linked to OpenStreetMap and built entirely on open-source software.

How it works

Our data pool connects anyone with real-time location data, processing, data storage, and routing and analysis applications.

Conscious of location data privacy, we allow contributors to share anonymized traffic speeds from GPS without any underlying information. In return, data contributors immediately gain access to routing and analysis tools.
